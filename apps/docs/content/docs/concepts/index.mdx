## Common Questions

### Do I need to understand all this to use KhipuVault?

**No.** You can use KhipuVault without deep technical knowledge, just like you can drive a car without understanding the engine.

However, understanding the basics will help you:
- Feel more confident
- Make better decisions
- Appreciate the security measures

### Is this like a bank?

**Not exactly.** Banks are centralized institutions that custody your money. KhipuVault is:
- Decentralized (no central authority)
- Non-custodial (you control your keys)
- Transparent (all code is open-source)

Think of it more like a **digital savings vault** you control.

### What if I don't understand blockchain?

That's okay! You don't need to understand blockchain deeply. Here's the simplest explanation:

> **Blockchain** = A shared ledger that everyone can see but no one can cheat.

All the complexity happens in the background. You just need to:
1. Connect your wallet
2. Deposit funds
3. Earn yields
4. Withdraw when you want

[Start here â†’](/docs/getting-started)

### Are there risks?

Yes, like any financial system. We cover:
- **Smart contract risks** in [Smart Contracts](/docs/concepts/smart-contracts)
- **Security measures** in [Security](/docs/concepts/security)
- **Market risks** in [Yield Generation](/docs/concepts/yield-generation)
