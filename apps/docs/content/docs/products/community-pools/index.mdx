# Community Pools

Save together, grow together. Create or join community-driven savings pools where groups pool resources, share yields, and make decisions collectively.

## What are Community Pools?

Community Pools are collaborative savings vehicles where groups of people deposit funds together, earn collective yields, and govern the pool democratically. Think of it as a credit union on blockchain.

**Key Features:**
- ü§ù **Group savings** - Pool resources with others
- üó≥Ô∏è **Democratic governance** - Vote on pool decisions
- üìà **Collective yields** - 12-18% APY on pooled funds
- üë• **Member management** - Add/remove members by vote
- üîç **Full transparency** - All transactions on-chain
- üí™ **Stronger together** - Bigger pools = better opportunities

## How It Works

```
Community ‚Üí Create Pool ‚Üí Members Join ‚Üí Deposit Together ‚Üí Earn Yields ‚Üí Govern Collectively
```

1. **Create or join a pool** - Start your own or join existing
2. **Deposit MUSD** - Each member contributes their share
3. **Yields accumulate** - Pool earns on total deposited amount
4. **Make decisions** - Vote on withdrawals, new members, strategies
5. **Withdraw when needed** - With member approval

## Pool Types

KhipuVault supports different pool configurations:

### Open Pools
- **Anyone can join** (subject to approval)
- Public discovery on platform
- Great for growing communities
- Example: Neighborhood associations

### Closed Pools
- **Invite-only** membership
- Private, not listed publicly
- Ideal for families and tight-knit groups
- Example: Extended family funds

### Hybrid Pools
- **Public with verification**
- Join requests require admin approval
- Balance between growth and security
- Example: Church congregations

## Governance Features

Community Pools include powerful governance tools:

- **Proposal System** - Members create and vote on proposals
- **Voting Mechanisms** - Democratic or weighted by contribution
- **Member Approval** - Vote to add/remove members
- **Withdrawal Requests** - Large withdrawals may require approval
- **Strategy Decisions** - Vote on yield strategies
- **Parameter Changes** - Adjust minimums, fees, rules
