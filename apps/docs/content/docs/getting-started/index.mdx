## Quick Start (5 Minutes)

If you want to jump right in:

Download and install [MetaMask](https://metamask.io/) browser extension.

### Add Mezo Network

Add Mezo testnet to your wallet with one click: [Add Mezo Network](/docs/getting-started/add-mezo-network)
