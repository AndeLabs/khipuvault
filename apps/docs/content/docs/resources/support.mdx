## Discord Support

**Best for:** General questions, quick help, community discussions

**Response Time:** Usually < 1 hour (community-driven)

**How to Get Help:**

1. **Join Discord**
   - [discord.gg/khipuvault](https://discord.gg/khipuvault)

2. **Choose the right channel:**
   - `#general` - General questions and discussions
   - `#support` - Technical help and troubleshooting
   - `#beginners` - New user questions
   - `#developers` - Development and API questions

3. **Ask your question clearly:**
   - What are you trying to do?
   - What happened instead?
   - Include error messages or screenshots
   - Mention your wallet address (if relevant)

### Discord Channels

| Channel | Purpose |
|---------|---------|
| `#announcements` | Official updates and news |
| `#general` | General chat and questions |
| `#support` | Technical support |
| `#beginners` | Help for new users |
| `#developers` | Developer discussions |
| `#feedback` | Product feedback and suggestions |
| `#bugs` | Report bugs |
| `#security-alerts` | Security announcements |

<Callout type="info" title="Discord Etiquette">
- Be respectful and patient
- Don't DM team members directly (use channels)
- Search before asking (your question may be answered)
- Provide details to get better help
- Thank those who help you
</Callout>

<Callout type="warning" title="Security Warning">
**NEVER share your private keys or seed phrase** in Discord or any public channel. Admins will NEVER DM you first. Be wary of scammers.
</Callout>

## Email Support

**Best for:** Non-urgent issues, detailed problems, account-specific questions

**Response Time:** < 4 hours (business hours), < 24 hours (weekends)

### General Support

**Email:** support@khipuvault.com

**Use for:**
- Transaction issues
- Deposit/withdrawal problems
- Balance discrepancies
- Product questions
- Bug reports

**Include in your email:**
- Clear subject line
- Wallet address
- Transaction hash (if applicable)
- Screenshots or error messages
- Steps to reproduce
- What you've already tried

**Example:**
```
Subject: Cannot withdraw from Individual Savings pool

Hi KhipuVault team,

I'm unable to withdraw from my Individual Savings pool.

Wallet: 0x1234...5678
Transaction hash: 0xabcd...
Error message: "Execution reverted: INSUFFICIENT_BALANCE"

I have 1,000 MUSD deposited and trying to withdraw 500 MUSD.

Screenshots attached.

Thank you!
```

### Technical Support

**Email:** bugs@khipuvault.com

**Use for:**
- Website bugs
- Smart contract issues
- Performance problems
- Integration errors

**Include:**
- Detailed bug description
- Steps to reproduce
- Expected vs. actual behavior
- Browser and OS version
- Console errors (if applicable)

### Documentation Feedback

**Email:** docs@khipuvault.com

**Use for:**
- Documentation errors
- Typos or outdated information
- Suggestions for new guides
- Translation requests

### Partnership & Business

**Email:** partnerships@khipuvault.com

## Security Support

**Best for:** Security vulnerabilities, suspicious activity, urgent threats

**Response Time:** < 1 hour (24/7)

### Report Security Issues

<Callout type="error" title="DO NOT report publicly">
**Never post security vulnerabilities** in Discord, GitHub issues, or Twitter. Use private channels only.
</Callout>

**Email:** security@khipuvault.com

**Use for:**
- Smart contract vulnerabilities
- Website security issues
- Suspected hacks or exploits
- Phishing attempts
- Suspicious transactions

**Eligible for bug bounty rewards** (up to $10,000)

[Learn more about our Bug Bounty Program](/docs/security/bug-bounty)

### Emergency Contacts

**Critical security incidents:**

1. **Email:** security@khipuvault.com (monitored 24/7)
2. **Discord:** Tag `@Security Team` in #security-alerts
3. **Twitter DM:** [@KhipuVault](https://twitter.com/khipuvault) (verified account only)

**What qualifies as emergency:**
- Active exploit of smart contracts
- Funds at risk
- Private key compromise
- Widespread phishing attack

### Report Suspicious Activity

**Phishing or scams:**
- Email: security@khipuvault.com
- Include screenshots and links
- Forward phishing emails
- Help protect the community

**Fake social media accounts:**
- Report to us: security@khipuvault.com
- Report to platform (Twitter, Discord, etc.)
- Include profile links and evidence

## Support Ticket Process

### What to Expect

1. **Acknowledge receipt** (< 1 hour during business hours)
2. **Initial assessment** (team member assigned)
3. **Investigation** (may ask for more details)
4. **Resolution or workaround** provided
5. **Follow-up** to ensure issue is resolved

### Response Expectations

**Priority levels:**

| Priority | Response Time | Resolution Time |
|----------|---------------|-----------------|
| Critical | < 1 hour | < 4 hours |
| High | < 4 hours | < 24 hours |
| Medium | < 24 hours | < 3 days |
| Low | < 48 hours | < 7 days |

**Critical:** Funds at risk, security issue, platform down

**High:** Cannot use core features, transaction failures

**Medium:** Inconvenient but workaround exists

**Low:** Questions, minor bugs, feature requests

### Escalation

**If you're not getting help:**

1. **Wait reasonable time** based on priority
2. **Bump your ticket** politely (after 24-48 hours)
3. **Try different channel** (Discord if email, vice versa)
4. **Escalate to team lead** (mention in Discord)

<Callout type="warning">
Please be patient. Our team is small but dedicated. We'll help everyone, but critical issues come first.
</Callout>
