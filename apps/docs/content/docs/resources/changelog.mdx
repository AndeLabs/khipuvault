## Breaking Changes

### v0.8.0 (January 25, 2026)

**API:**
- `GET /pools` response structure changed
- Now includes `poolType` field
- Old response no longer supported

**Migration:**
```javascript
// Old
const pools = await fetch('/api/pools')
// pools: [{ id, name, tvl }]

// New
const pools = await fetch('/api/pools')
// pools: [{ id, name, tvl, poolType, members }]
```

---

### v0.7.0 (January 10, 2026)

**Contracts:**
- `deposit()` function signature changed
- Now requires `poolId` parameter

**Migration:**
```solidity
// Old
deposit(amount)

// New
deposit(poolId, amount)
```
