# @khipu/ui

Shared UI components for KhipuVault monorepo.

Built with:
- Radix UI primitives
- Tailwind CSS
- class-variance-authority

## Usage

```tsx
import { Button, Card } from '@khipu/ui'

export function MyComponent() {
  return (
    <Card>
      <Button>Click me</Button>
    </Card>
  )
}
```

## Development

```bash
pnpm build
pnpm dev
```
