# cspell Quick Reference

## Commands

```bash
# Check all files
pnpm spell:check

# Check only staged files
pnpm spell:check:changed

# Check specific files
pnpm exec cspell "apps/web/**/*.tsx"
```

## Adding Words

Edit `.cspell-custom.txt`:

```bash
# Add your word alphabetically in the appropriate section
nano .cspell-custom.txt
```

## Inline Ignore

```typescript
// Single line
// cspell:disable-next-line
const myWeirdVar = "xyz";

// Block
/* cspell:disable */
// Code with special terms
/* cspell:enable */

// Entire file
/* cspell:disable */
```

## Common Sections in .cspell-custom.txt

- Blockchain & DeFi Terms
- Solidity & Smart Contract Terms
- Tech Stack & Tools
- Libraries & Packages
- Abbreviations & Acronyms
- Contract Names

## Pre-commit Hook

Automatically runs on markdown files only during `git commit`.

To bypass: `git commit --no-verify`
